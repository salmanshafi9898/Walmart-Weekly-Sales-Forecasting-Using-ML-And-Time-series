# Walmart-Weekly-Sales-Forecasting-Using-ML-And-Time-series
 Built end-to-end forecasting system using a 6,435-observation dataset across 45 stores and 143 weeks; validated key demand drivers via PSM and Lasso methods, benchmarked five models with 10-fold CV, and deployed Random Forest and SARIMAX to deliver automated, data-driven inventory and staffing forecasts.
